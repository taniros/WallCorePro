# Morning & Night Wishes – Feature Suggestions

Ideas to consider for future releases. Prioritize based on user feedback and metrics.

---

## ✅ Done
- **Internet connectivity notice** – Banner prompts users to check Wi‑Fi/mobile data when offline
- 16 KB page size support
- Crashlytics mapping upload disabled for placeholder config

---

## 🚀 High-impact features

### 1. **Offline mode**
- Cache favorite wallpapers for offline viewing
- Show cached wallpapers when offline instead of empty state
- "Download for offline" on detail screen

### 2. **Schedule wishes**
- Set time to send a wish to a contact (e.g. 7 AM)
- Reminder to send birthday / occasion wishes

### 3. **Wish templates**
- Saved favorite wishes for quick sharing
- History of sent wishes

### 4. **Social sharing**
- Direct share to WhatsApp / Instagram status
- One-tap share with pre-filled wish

### 5. **Widget**
- Home-screen widget with daily wish or wallpaper
- Quick “share today’s wish” action

---

## 📈 Engagement & retention

### 6. **Streaks & goals**
- “Share X wishes this week” goals
- Streak badges / milestones

### 7. **Personalization**
- Learn preferred quote styles and categories
- Time-of-day preferences (morning vs night user)

### 8. **Notifications**
- Optional “send a wish” reminder at a chosen time
- Occasion reminders (e.g. Mother’s Day)

### 9. **In-app calendar**
- Occasions with suggested wishes
- Birthday import from contacts (with permission)

---

## 🎨 UX improvements

### 10. **Dark/light toggle**
- Per-screen or system-based theme
- AMOLED black option

### 11. **Font size**
- Adjustable text size in wishes

### 12. **Multi-language**
- Localization for major markets (e.g. Hindi, Spanish, Arabic)

### 13. **Accessibility**
- TalkBack improvements
- High-contrast options

---

## 🔧 Technical

### 14. **Analytics**
- Track which wishes/categories are most used
- A/B test different wish styles

### 15. **Backup**
- Restore favorites after reinstall
- Optional cloud sync for saved wishes

### 16. **Performance**
- Preload next wallpaper in detail view
- Reduce image size for faster loading

---

*Update this list as features ship or priorities change.*
